
根据 product_manual.md 文档，以下是 MVP 阶段需要优先实现的重要功能：

1. **智能扫描**模块中的关键功能：
   - 智能识别常见垃圾文件类型（缓存、临时文件等）
   - 支持基础的自定义扫描规则
   - 简化版扫描结果展示

2. **一键清理**功能的核心部分：
   - 支持一键清理所有扫描出的垃圾文件
   - 清理前提供预览功能
   - 基础的清理报告生成

3. **安全保障**中的必要功能：
   - 清理前备份重要文件的基础实现
   - 完善内置白名单机制，防止误删系统文件

4. **多平台支持**：
   - 扩展到 Windows 平台支持（目前主要针对 macOS）

5. **其他必要功能**：
   - 扫描/清理进度实时展示
   - 基础通知提醒功能

这些功能是在现有已实现功能（如定时清理、文件系统操作工具）基础上，使产品形成完整闭环的必要补充，应在 MVP 阶段优先开发。
